# 📘 Panduan Penggunaan EduDoc Pro
**Alat Bantu PDF Mandiri & Aman untuk Dosen & Akademisi**

EduDoc Pro adalah solusi praktis untuk mengelola administrasi dokumen tanpa rasa khawatir. Seluruh pemrosesan berkas dilakukan **100% di perangkat Anda (Client-Side)**. Data Anda tidak pernah dikirim ke internet.

---

## 🚀 Cara Mengakses Aplikasi

Bapak/Ibu dapat menggunakan aplikasi ini melalui dua jalur berikut:

### Jalur 1: Akses Cepat via Web (Rekomendasi)
*Cocok untuk penggunaan langsung tanpa unduh file.*

1. **Buka Tautan:** Akses di [LIVE ON SERVER] (https://sutanto-teguh.github.io/edudoc/)
2. **Instalasi (PWA):**
   - **Desktop (Chrome/Edge):** Klik ikon **Install** (gambar monitor) pada bar alamat atau klik tombol **"Install Sekarang"** di dalam aplikasi.
   - **HP (Android/iOS):** Pilih menu **"Tambahkan ke Layar Utama"** pada pengaturan browser.
3. **Keunggulan:** Aplikasi akan selalu mendapatkan pembaruan otomatis dan dapat dibuka meski tanpa koneksi internet setelah terinstal.

### Jalur 2: Salinan Mandiri (via Google Drive)
*Cocok jika Anda ingin menyimpan file master aplikasi di penyimpanan pribadi.*

1. **Download:** Unduh file `EduDoc_Pro.html` dari folder [Google Drive](https://drive.google.com/drive/folders/1RS6oz5Ml2ypj_VHGEeDC5TgxKXBZp_IB?usp=drive_link)  yang tersedia.
2. **Jalankan:** Klik kanan pada file tersebut, lalu pilih **Open With > Google Chrome** atau **Microsoft Edge**.
3. **Instalasi Manual:**
   - Klik **Menu Browser (titik tiga ⋮)** di pojok kanan atas.
   - Pilih **"Install EduDoc Pro..."** atau **"Save and Share" > "Install as App"**.
4. **Keunggulan:** File ini sepenuhnya milik Anda dan bisa dibagikan secara offline via flashdisk.

## 🛠️ Fitur Utama

| Fitur | Deskripsi | Cara Pakai |
| :--- | :--- | :--- |
| **Gabung PDF** | Menyatukan beberapa file menjadi satu dokumen. | Pilih file-file PDF, cek urutan, klik **Gabungkan**. |
| **Pisah PDF** | Mengambil halaman tertentu dari file PDF. | Masukkan file, ketik nomor halaman (misal: 1-5), klik **Ekstrak**. |
| **Kompres PDF** | Memperkecil ukuran file tanpa merusak kualitas signifikan. | Pilih file besar, tunggu analisis, klik **Kompres**. |

## 🛡️ Keamanan & Privasi Data

Privasi Bapak/Ibu adalah prioritas kami:
- **Zero Server Upload:** Berkas Bapak/Ibu **TIDAK AKAN PERNAH** diunggah ke server mana pun.
- **Offline First:** Setelah aplikasi dimuat atau diinstal, internet tidak lagi diperlukan.
- **Bebas Pelacakan:** Tidak ada sistem login, tidak ada iklan, dan tidak ada pengumpulan data aktivitas.
